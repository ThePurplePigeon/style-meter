# Security

Style Meter is a local Dalamud plugin. It does not use a backend service.

## Reporting

Report security issues privately through GitHub security advisories if available. If that is not available, contact the maintainer before opening a public issue.

## Scope

Security reports should include:

- Crashes caused by malformed or unexpected game data.
- Unsafe hook lifecycle behavior.
- Accidental network, telemetry, or data persistence behavior.

Gameplay suggestions and normal bugs should use GitHub issues.
